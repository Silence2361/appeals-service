# Appeals System

REST API для управления обращениями пользователей: создание, взятие в работу, завершение, отмена, массовая отмена и получение списка с фильтрацией по дате.

---

## Стек технологий

- Node.js + TypeScript
- Express.js
- MongoDB + Mongoose

---

## Установка

npm install

---

## Запуск

npm run start
